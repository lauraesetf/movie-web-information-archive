# movie-web-information-archive
该仓库主要用于收集和归档影视相关网站的公开基础信息，覆盖电影、电视剧、综艺及动漫等不同类型站点。整理内容仅限站点层级的客观资料，不包含任何影视内容文件或资源链接。所有资料均整理自公开渠道，目的在于形成清晰、可长期维护的站点信息档案，方便后续核对、对照与查询。本项目不参与站点运营，也不对站点内容质量、版权属性或更新频率作出判断，仅作为中性的信息整理与资料归档用途存在。
